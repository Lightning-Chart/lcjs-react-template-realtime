Minimal LightningChart JS + React + Websocket application for real-time multi-channel line chart visualization

Create `frontend/.env` and define REACT_APP_LCJS_LICENSE="your license key here"

`cd backend`
`npm i`
`node index`

In another terminal:

`cd frontend`
`npm i`
`npm start`
